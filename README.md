# gao-shanwen-research-model

![纪念高善文老师：我做了一个整理他著作与文章的 Skill](assets/wechat-cover.png)

一个用于整理、检索和学习高善文老师公开著作与文章的 Codex Skill。

## 缘起

高善文老师离世后，我想为自己留下一份可以反复使用的资料入口。

我也是高老师的读者和粉丝。刚入行时读到他的《上升的地平线——刘易斯拐点与通货膨胀》，第一次强烈感到宏观研究可以既严谨、克制，又有穿透力：不是堆砌概念，而是在纷繁数据中寻找可以被验证、也可以被证伪的因果链条。

这个 Skill 是出于纪念、学习和资料整理而做的。它不是让 AI “替高老师回答问题”，也不试图模仿高老师的口吻下结论。高老师的判断、文字和研究，只能回到他公开发表过的著作和文章中去理解。

## 高善文老师简介

高善文老师，1971年9月出生于山西临汾。许多人认识他，是从资本市场上那位以逻辑、数据和独立判断著称的首席经济学家开始；但他的起点，是山西南部丘陵地区的一个乡村。这样的出身，使他后来的文字里始终有一种具体的现实感：不是从概念出发解释中国，而是从生活经验、制度约束和可观察的数据中寻找因果链条。

1988年，着迷于物理学的高善文考入北京大学无线电电子学专业，开始了七年燕园求学。北大的讲座、思想争论和学术空气，对他影响很深。他后来在《时光的刻痕》中回忆，北大像一个流动着各种新思想的“自由市场”。之后，他从电子学转向经济学，进入北京大学国民经济管理系，也就是今天北大光华管理学院的前身，并在1995年获得经济学硕士学位。

他的代表著作包括《在周期的拐点上》《透视繁荣：资产重估深处的忧虑》《经济运行的逻辑》《经济运行的真相》等。2026年7月7日，高善文老师因病离世，享年55岁。对于我这样曾经读着他的文章入行的人来说，整理他的著作和文章，不是为了让 AI 替他回答问题，而是希望更多人能回到他的文字里，重新理解那种严谨、克制、敢于被证伪的宏观研究。

## 这个 Skill 是什么

`gao-shanwen-research-model` 整理了三类内容：

1. 高善文老师公开出版或参与的书籍、版本和书目信息。
2. 高善文纪念站中公开整理的文章索引，包括宏观研究、市场评论、随笔和代表性文章。
3. 从高老师公开研究中提炼出的分析流程：现象、机制、数据交叉验证、情景推演和证伪条件。

它更像一个研究入口和阅读地图，而不是一个观点生成器。

## 它能做什么

- 查找高老师有哪些著作、版本和推荐阅读顺序。
- 根据主题寻找相关公开文章，例如刘易斯拐点、通货膨胀、资产重估、房地产、人民币汇率、影子银行、中美贸易摩擦等。
- 帮助整理高老师研究方法的脉络。
- 在分析中国宏观经济和资本市场问题时，提醒使用者先找事实、再找机制、再做数据验证。
- 生成读书笔记、文章导读、研究 memo 或资料索引。

## 它不做什么

- 不冒充高老师本人。
- 不声称“高老师现在会怎么看”。
- 不把 AI 的输出包装成高老师的观点。
- 不替代原文阅读。
- 不把宏观或市场判断当成投资建议。

如果问题涉及当下宏观经济、政策或市场价格，仍然需要重新检索最新数据，并注明数据日期和来源。

## 资料范围

当前 Skill 仅整理公开书目信息、公开文章索引和阅读线索，不收录书籍正文、完整文章或受版权保护的下载资源。

当前 Skill 收录并整理了：

- 《选好公司 买对股票：2005中国经济展望与行业投资策略》
- 《在周期的拐点上：从数据看中国经济的波动》
- 《透视繁荣：资产重估深处的忧虑》
- 《踏遍青山牛未老：2008年中国A股投资策略》
- 《经济运行的逻辑》
- 《经济运行的真相》
- 高善文纪念站公开文章索引 108 条

其中部分书籍存在版本、出版机构或页数信息差异，Skill 中保留了这些差异，并要求使用时交叉验证。

## 使用方式

在 Codex 中可以这样调用：

```text
用 $gao-shanwen-research-model 给我整理高善文老师的著作阅读顺序。
```

```text
用 $gao-shanwen-research-model 找一下高善文老师关于刘易斯拐点和通胀机制的文章。
```

```text
用 $gao-shanwen-research-model 分析当前中国房地产下行对 A 股估值的影响，注意不要替高老师发言，只使用他的研究框架。
```

## 一个基本原则

尊重高老师，最好的方式不是让 AI 复述一个确定答案，而是回到他的文字、问题意识和研究方法：

先把事实看清楚，再把机制想明白；

先承认证据的边界，再提出可以被检验的判断。

这个 Skill 只是为这件事做一点整理工作。
